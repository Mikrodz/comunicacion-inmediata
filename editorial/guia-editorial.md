# Guía editorial

## Propósito

Esta guía establece los criterios editoriales que rigen toda la colección *Comunicación inmediata*.

Su objetivo es definir qué contenido forma parte de la colección, cómo se organiza y bajo qué criterios se selecciona.

# Principios editoriales

## Comunicación antes que idioma

La prioridad es permitir que dos personas puedan comprenderse.

La teoría lingüística nunca debe convertirse en una barrera para la comunicación.

## Utilidad inmediata

Todo el contenido debe poder utilizarse en una situación real el mismo día en que el lector consulta la guía.

## Contexto antes que traducción

Las expresiones siempre se presentan dentro de un escenario concreto.

No existen traducciones universales.

Una misma idea puede expresarse de formas diferentes según el contexto.

## Realismo

Solo se incluyen expresiones utilizadas de manera natural por hablantes reales.

Las traducciones literales o poco frecuentes se descartan.

## Aprendizaje incidental

Aunque la colección no constituye un curso de idiomas, cada escenario debe favorecer el aprendizaje natural mediante la repetición de patrones frecuentes.

El aprendizaje surge como consecuencia del uso.

No como requisito previo.

## Modularidad

Todo el contenido debe poder reutilizarse sin modificaciones sustanciales en distintos formatos:

- PDF
- impresión
- sitio web
- aplicación móvil
- tarjetas rápidas
- simulaciones
- recursos multimedia

# Organización del contenido

La unidad principal de la colección es el **escenario**.

Cada escenario representa una situación concreta y puede incluir:

- microdiálogos
- frases individuales
- patrones reutilizables ("Construí nuevas frases")

# Criterios de inclusión

Una expresión forma parte de la colección cuando:

- resuelve una necesidad real de comunicación
- resulta natural para hablantes del contexto correspondiente
- aporta información útil
- evita duplicar expresiones equivalentes
- puede localizarse rápidamente durante una consulta

# Criterios de exclusión

No forman parte de la colección:

- listas extensas de vocabulario sin contexto
- explicaciones gramaticales desarrolladas
- traducciones literales poco naturales
- expresiones artificiales o infrecuentes
- contenido que no aporte utilidad inmediata
