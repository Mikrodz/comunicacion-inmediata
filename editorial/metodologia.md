# Metodología de investigación

## Objetivo

Identificar patrones reales de comunicación para diseñar escenarios reutilizables.

## Marco de referencia

Este proyecto utiliza el Marco Común Europeo de Referencia para las Lenguas (MCER) como referencia para estimar la complejidad lingüística de las expresiones y mantener la consistencia editorial.

El MCER no determina la estructura del manual ni la secuencia de los contenidos. Los escenarios se organizan según necesidades de comunicación y no según objetivos gramaticales.

## Público objetivo

El manual está dirigido a personas que necesitan comunicarse en otro idioma para resolver una situación concreta, pero que aún no poseen la fluidez suficiente para hacerlo con comodidad.

Como referencia editorial, el contenido está orientado principalmente a lectores con un dominio aproximado entre A0 y A2 según el MCER.

Se asume que el lector puede reconocer palabras y expresiones frecuentes, pero aún no posee la fluidez suficiente para mantener conversaciones espontáneas.

Las expresiones seleccionadas deben poder comprenderse y utilizarse con una carga cognitiva baja, priorizando la comunicación inmediata por sobre la complejidad gramatical.

## Fuentes

Las expresiones no se obtienen de una única fuente.

Cada escenario debe sustentarse en varias evidencias.

Según el dominio, las fuentes pueden incluir:

### Bibliografía

Artículos, investigaciones y publicaciones especializadas.

### Documentación

Guías, manuales, documentación oficial y material de referencia.

### Comunidad

Foros, Reddit, Discord y otros espacios donde interactúe la comunidad.

### Material audiovisual

Videos, transmisiones, grabaciones, entrevistas o cualquier otro contenido donde aparezcan situaciones reales de comunicación.

Cuando resulte útil, este material puede procesarse mediante herramientas de transcripción automática (por ejemplo, Whisper) para facilitar su análisis.

Las transcripciones constituyen únicamente material de investigación y no forman parte del contenido final del manual.

### Observación directa

Cuando sea posible, se podrán observar situaciones reales de comunicación dentro del propio dominio.

## Pipeline de investigación

↓

↓

↓

↓

↓

↓

Construir nuevas frases

## Criterios de inclusión

Una expresión se incorpora únicamente cuando:

- resuelve una necesidad concreta de comunicación
- aparece respaldada por varias evidencias
- resulta comprensible para personas con un nivel inicial de inglés (A0–A2 según el MCER)
- puede reutilizarse en diferentes situaciones del mismo dominio
- mantiene un lenguaje claro y natural

## Criterios de exclusión

Una expresión no se incorpora cuando:

- depende exclusivamente de un caso muy específico
- requiere conocimientos avanzados del idioma
- aparece únicamente en una fuente sin evidencia adicional
- puede reemplazarse por otra expresión más simple con el mismo significado
- no aporta valor para resolver la necesidad de comunicación del escenario
