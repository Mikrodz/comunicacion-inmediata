# Guía de estilo

## Propósito

Esta guía define las convenciones utilizadas para presentar el contenido de manera consistente en toda la colección.

## Estilo de redacción

El lenguaje debe ser:

- claro
- breve
- natural
- contextual
- fácil de consultar

Siempre que existan varias alternativas correctas, se priorizará la expresión más utilizada por hablantes reales dentro del contexto correspondiente.

## Estructura de un escenario

Cada escenario puede contener:

- una breve introducción contextual (opcional)
- uno o más microdiálogos
- frases individuales
- una sección **Construí nuevas frases**

## Tono

Cuando resulte necesario, una expresión puede incluir una indicación de tono.

Las categorías iniciales son:

- 😊 Amigable
- ⚠️ Brusco
- 🎩 Muy formal

Las etiquetas solo deben utilizarse cuando aporten información relevante para el lector.

## Lectura aproximada

Cada expresión en lengua extranjera incluye una **(🔊 lectura aproximada)** orientada a hispanohablantes.

La lectura aproximada no constituye una representación fonética académica ni pretende reemplazar el Alfabeto Fonético Internacional (AFI).

La colección utiliza una convención editorial propia basada en los siguientes principios:

- utilizar letras familiares para hispanohablantes
- evitar símbolos especializados
- introducir la menor cantidad posible de convenciones nuevas

Siempre se presenta utilizando el siguiente formato:

> Heal me.
> (🔊 jiil mi)

## Construí nuevas frases

Cuando un escenario lo permita, se presentarán patrones reutilizables que ayuden al lector a producir nuevas expresiones.

Estos patrones no buscan enseñar gramática de forma explícita.

Su objetivo es favorecer el aprendizaje incidental mediante el uso repetido de estructuras frecuentes.

## Convenciones tipográficas

Se utilizarán convenciones consistentes en toda la colección para facilitar la consulta rápida.

Las decisiones de diseño deben priorizar siempre:

- legibilidad
- accesibilidad
- consistencia
- rapidez de consulta
